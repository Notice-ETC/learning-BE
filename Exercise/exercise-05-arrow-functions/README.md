# Exercise 05: Arrow Functions

## Objective
Learn to use arrow functions (modern JavaScript/TypeScript syntax).

## Instructions
1. Create arrow function to subtract two numbers
2. Create arrow function to divide two numbers
3. Create arrow function to check if number is even
4. Create arrow function to convert Celsius to Fahrenheit
5. Call all functions and print results

## Expected Output
```
Difference: 5
Quotient: 2
Is Even: true
Fahrenheit: 77
```

## Hints
- Arrow function syntax: `const name = (param: type): returnType => { }`
- For single expression: `const name = (param: type): returnType => expression`
- Formula: Fahrenheit = (Celsius Ã— 9/5) + 32
