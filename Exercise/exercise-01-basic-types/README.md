# Exercise 01: Basic Types

## Objective
Learn to declare variables with proper TypeScript types.

## Instructions
1. Create variables with the following types:
   - A string variable for a person's name
   - A number variable for age
   - A boolean variable for student status
   - A null variable
   - An undefined variable

2. Try to assign wrong types and see TypeScript errors

## Expected Output
```typescript
console.log(personName)      // "John Doe"
console.log(age)       // 25
console.log(isStudent) // true
```

## Hints
- Use `let` or `const` for variable declaration
- Specify type after colon: `variableName: type`
