# Exercise 03: String Operations

## Objective
Learn to manipulate strings in TypeScript.

## Instructions
1. Create string variables for first name and last name
2. Concatenate them to create full name
3. Convert full name to uppercase
4. Convert full name to lowercase
5. Get the length of the full name
6. Extract first 5 characters

## Expected Output
```
Full Name: John Doe
Uppercase: JOHN DOE
Lowercase: john doe
Length: 8
First 5 characters: John 
```

## Hints
- Use `+` or template literals for concatenation
- Use `.toUpperCase()` and `.toLowerCase()` methods
- Use `.length` property
- Use `.substring()` or `.slice()` for extraction
