# Exercise 06: Interfaces

## Objective
Create and use interfaces to define object structures.

## Instructions
1. Create interface for a Person (name, age, email)
2. Create interface for a Product (id, name, price, inStock)
3. Create objects using these interfaces
4. Create function that takes Person and returns greeting
5. Create function that calculates total price for products

## Expected Output
```
Person: John Doe, 30, john@example.com
Greeting: Welcome, John Doe!
Product: Laptop ($999.99)
Total Price: $1299.98
```

## Hints
- Interface syntax: `interface Name { property: type }`
- Use interface as type: `const obj: InterfaceName = { }`
