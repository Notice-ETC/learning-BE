# Exercise 02: Number Operations

## Objective
Perform basic arithmetic operations with TypeScript number types.

## Instructions
1. Create two number variables
2. Perform the following operations:
   - Addition
   - Subtraction
   - Multiplication
   - Division
   - Modulo (remainder)
3. Store results in properly typed variables

## Expected Output
```
Sum: 30
Difference: 10
Product: 200
Quotient: 2
Remainder: 0
```

## Hints
- Use arithmetic operators: +, -, *, /, %
- Declare result variables with `: number` type
