# Exercise 04: Functions

## Objective
Create and use regular functions with proper TypeScript typing.

## Instructions
1. Create a function that adds two numbers
2. Create a function that multiplies two numbers
3. Create a function that greets a person by name
4. Create a function that calculates circle area (πr²)
5. Call all functions and print results

## Expected Output
```
Sum: 15
Product: 50
Greeting: Hello, Alice!
Circle Area: 78.54
```

## Hints
- Function syntax: `function name(param: type): returnType { }`
- Use `Math.PI` for π
- Use `Math.pow(r, 2)` or `r * r` for r²
